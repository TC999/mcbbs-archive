高中三年和同学-联机一起玩，建的一个小城市。城市中间是一个巨大的城堡。旁边有各种居民楼。公路一些建筑的沙漠上建了一个巨大的金字塔，还有一些名胜古迹。城市有一条轻轨贯穿整个城市。虽然每一个建筑的做工可能不是那么的精致，但是这是我读高中三年留下的美好的回忆，现在我将它分享给大家。
